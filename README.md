# TG-Proxy-Parser
 Python multithreading parser that parses Vmess, Vless, Reality, Trojan, ShadowSocks, Hysteria-1, Hysteria-2, Tuic, Juicity, Naive, Nekoray and Socks(4-5) proxy configs from telegram channels.
